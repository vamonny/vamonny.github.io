Вопросы

Какие сроки выполнения?

Логотип — 3-5 дней. Фирменный стиль — 2-3 недели. Инфографика — 2-3 дня. 3D анимация — 5-7 дней. Срочные заказы обсуждаются отдельно.

Сколько правок включено?

2-3 раунда правок включены в стоимость. Обычно этого достаточно. Дополнительные правки — по договорённости.

В каких форматах получу файлы?

AI, SVG, PNG, JPG, PDF — все основные форматы. Для печати — CMYK, для веба — RGB. Исходники в Adobe Illustrator.

Нужна ли предоплата?

Для новых клиентов — 50% предоплата. Для постоянных — по договорённости. Оплата переводом на карту.

Работаете с регионами?

Да, работаю удалённо с клиентами из любого города. Связь через Telegram, WhatsApp или email.
